# Selenium Python Automation Practice

## Overview
This repository contains beginner-friendly Selenium automation scripts developed using Python.

## Concepts Covered
- Browser Automation
- Element Locators
- Link Navigation
- Dynamic Tables
- Loops
- Explicit Waits

## Files

### 01_open_browser.py
Launches Chrome and opens Google.

### 02_click_link.py
Clicks the "Sortable Data Tables" link.

### 03_dynamic_table.py
Reads rows from a dynamic table.

### 04_explicit_wait.py
Demonstrates the use of Explicit Waits.

### 05_dynamic_table_validation.py
Counts table rows and prints all table data using loops.

## Tools Used
- Python
- Selenium WebDriver
- Google Chrome
- Visual Studio Code

## How to Run

```bash
pip install selenium
python filename.py
```
